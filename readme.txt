1. Задача — написать функцию, которая фильтрует товары по переданным критериям. Например:

const filters = { minPrice: 50, maxPrice: 200, category: "electronics" };

Эта функция должна вернуть только товары из категории "electronics" с ценой от 50 до 200.

2. Напиши функцию, которая принимает два объекта и проверяет, равны ли они по содержимому (включая вложенные объекты). Например, { a: { b: 2 } } и { a: { b: 2 } } должны считаться равными.
